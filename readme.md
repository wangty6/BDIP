# Basic Digital Image Processor
## Intro
A naive image processor build for DIP course in 2016-2017 semester.
## Function
now avaliable:
* List Image info (color space, size and so on)
* Scale in size
* Quantization
* Historgram Equalization
* Space Filter by Convolution
